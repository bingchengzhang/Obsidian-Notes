## 1. 常用命令矩阵

| 维度       | 指令 (需加 `.\`)                     | 功能描述               | AI 专业使用场景                     |     |
| -------- | -------------------------------- | ------------------ | ----------------------------- | --- |
| **课程查询** | `lazy course list`               | 列出所有课程及对应的 **ID**  | 快速找“离散”或“FDS”的 ID             |     |
| **文件查询** | `lazy file list --id [ID]`       | 列出该课程下所有可下载文件      | 下载前确认课件更新到哪了                  |     |
| **下载课件** | `lazy course download --id [ID]` | **全量下载**该课程的所有课件   | 学期初一键同步所有 PDF                 |     |
| **配置查看** | `lazy config show`               | 查看当前的 Cookie 和下载路径 | 确认 `iPlanetDirectoryPro` 是否生效 |     |
| **配置备份** | `lazy config backup`             | 导出当前配置文件的路径        | 找不到 `yaml` 时用来定位              |     |
| **版本检查** | `lazy --version`                 | 查看当前工具版本           | 报错时用来排查兼容性                    |     |

---

## 2. 探索“隐藏命令”的秘诀

如果上面的命令报错（比如 `No such command`），说明你的版本层级更深。作为 AI 专业学生，最核心的技能是**看文档**。

在命令行里输入以下指令，它会告诉你当前版本**真正**支持什么：

- `.\lazy.exe --help` （查看顶级命令）
    
- `.\lazy.exe course --help` （查看课程相关的子命令，**下载命令可能藏在这里**）
    
- `.\lazy.exe config --help` （查看配置相关的子命令）
